# MyPrivateNetwork

Система управления Xray VPN сервером

## Возможности
- Создание и управление пользователями Xray
- Автоматическая настройка конфигурации сервера
- Мониторинг подключений
- REST API для управления пользователями

## Технологии
- Golang
- Xray-core
- SQLite/PostgreSQL
- Docker
